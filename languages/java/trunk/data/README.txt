Some example PDUs generated by NPS's AUV Workbench. These are entity state PDUs
with articulation parameters. 

The data was generated by open-dis 3.0, so its value as an interoperability test suite
for open-dis 3.0 is questionable. But it's useful to have some PDUs around.

disbin is IEEE DIS PDUs concatenated togeter in binary format, one PDU after another.
the XML file is the same data, only represented in XML.
